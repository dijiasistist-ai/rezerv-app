# zuvu

zuvu marketplace ve isletme paneli prototipi.

## Local gelistirme

```bash
npm install
npm start
```

Local adres:

```text
http://127.0.0.1:8091/
```

Isletme paneli:

```text
http://127.0.0.1:8091/venue.html
```

Demo hesaplar:

- `demo@zuvu.app` / `123456`
- `firma@zuvu.app` / `123456`
- `admin@zuvu.app` / `123456`

## Render deploy

Render Web Service ayarlari:

- Build command: `npm install`
- Start command: `npm start`
- Runtime: Node

Repo kokundeki `render.yaml` blueprint olarak ayni ayarlari icerir.
